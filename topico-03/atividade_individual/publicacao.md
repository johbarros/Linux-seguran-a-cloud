# Publicação do serviço web

## Nível escolhido
Nível 1 - Essencial

## Rota escolhida
Apache HTTP Server

## Página criada
index.html

## Local de publicação
/var/www/html/

## Comandos principais utilizados
- sudo apt install apache2
- sudo cp index.html /var/www/html/
- sudo systemctl restart apache2

## Resultado obtido
O serviço web foi publicado com sucesso e a página ficou acessível através do navegador.

## Limitações encontradas
Nenhuma limitação significativa. Apenas necessidade de verificar permissões e reiniciar o serviço.
