# Tópico 1 - Preparação do ambiente Linux

## Ambiente utilizado

Linux Mint instalado localmente no computador.

## Objetivo desta atividade

Preparar o ambiente Linux e organizar a primeira evidência técnica.

## Estrutura criada

Foi criada uma estrutura de diretórios para armazenar evidências, comandos executados e documentação técnica do módulo.

## Comandos executados

* pwd → mostra o diretório atual
* whoami → mostra o utilizador atual do sistema
* hostname → mostra o nome da máquina
* uname -a → mostra informações do sistema Linux
* df -h → mostra o espaço disponível em disco
* free -h → mostra a memória RAM disponível

## Diferença entre VM, VPS e infraestrutura em nuvem

VM:
Máquina virtual executada localmente no computador.

VPS:
Servidor virtual hospedado na internet.

Infraestrutura em nuvem:
Conjunto de serviços e servidores fornecidos por plataformas cloud como AWS, Azure e Google Cloud.
